## ğŸ› ï¸ Run Locally

**Prerequisites:** Node.js (v18 or higher)

1. **Clone the repository**
   ```bash
   git clone https://github.com/ItsNecron/Dev-Portfolio.git
   cd Dev-Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```
